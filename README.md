# react-typescript
This is a basic react project that demonstrate typescript
Topics covered :
  - basic and advanced props
  - Events & Styles
  - useState
  - useReducer
  - ContextAPI
  - useRef
  - Class component
  - Components as props
  - Generics
  - Restricting props
  - Template literals & Exclude
  - Wrapping HTML & Omit
  - Extracting component prop types
  - Polymorphic components
